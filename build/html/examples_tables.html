Examples: LaTeX Tables¶

Note

All of these examples assume that your document is employing the nasa-latex-docs class which loads all the necessary packages and defines the format settings. You should be able to recreate all of these examples by simply copying and pasting the code snippets below into your document .tex file.

Outside of the nasa-latex-docs class, all of these examples maybe recreated by simply including the same package list as defined in nasa-latex-docs/support/packages/packages.tex. An example is provided below:

\documentclass[12pt]{article}

% document preamble
\input{nasa-latex-docs/support/packages/packages.tex}

\begin{document}

% document content

\end{document}

Table with Caption¶

Text

\begin{table}[H]
   \caption{Example LaTeX Table} 
   \label{tab:TabEx}
   \small % text size of table content
   \centering % center the table
   \begin{tabular}{lccr} % alignment of each column data
   \toprule[\heavyrulewidth]\toprule[\heavyrulewidth]
   \textbf{Header 1} & \textbf{Header 2} & \textbf{Header 3} & \textbf{Header 4} \\ 
   \midrule
   Item 1 & Item 2 & Item 3 & Item 4 \\
   Item 1 & Item 2 & Item 3 & Item 4\\
   \bottomrule[\heavyrulewidth] 
   \end{tabular}
\end{table}

Table Using \multicol{}¶

Text

\begin{table}[H]
   \caption{Example LaTeX Table with \texttt{\textbackslash multicolumn}} 
   \label{tab:TabEx}
   \small % text size of table content
   \centering % center the table
   \begin{tabular}{lccr} % alignment of each column data
   \toprule[\heavyrulewidth]\toprule[\heavyrulewidth]
   \textbf{Header 1} & \textbf{Header 2} & \textbf{Header 3} & \textbf{Header 4} \\ 
   \midrule
   \multicolumn{3}{c}{Multiple Column Entry} & Item 4 \\
   Item 1 & Item 2 & Item 3 & Item 4\\
   \bottomrule[\heavyrulewidth] 
   \end{tabular}
\end{table}

Table Using \multirow{}¶

Text

\begin{table}[H]
   \caption{Example LaTeX Table with \texttt{\textbackslash multirow}}  
   \label{tab:TabEx}
   \small % text size of table content
   \centering % center the table
   \begin{tabular}{lccr} % alignment of each column data
   \toprule[\heavyrulewidth]\toprule[\heavyrulewidth]
   \textbf{Header 1} & \textbf{Header 2} & \textbf{Header 3} & \textbf{Header 4} \\ 
   \midrule
   \multirow{2}{*}{Multiple Row Entry} & Item 2 & Item 3 & Item 4 \\
                                       & Item 2 & Item 3 & Item 4\\
   \bottomrule[\heavyrulewidth] 
   \end{tabular}
\end{table}
